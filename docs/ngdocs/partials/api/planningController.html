planningController

Description

Planning controller

Methods

  • add1day()

    Go to next day

  • gototoday()

    Go to current day

  • initialize()

    Controller initialization :

    • Get local mission(s)
    • Reduce mission.assets array considering pending reports ($scope.removeObsoleteMission())
    • Send pending missions related reports (TODO)
    • Get remote mission(s) ($scope.synchronize())
    • Set current day : today at midnight or last viewed day
    • Initialize counts ($scope.updateCount())

  • remove1day()

    Go to preview day with mission(s)

  • removeObsoleteMission()

    Reduce mission.assets array considering pending reports

  • synchronize()

    Get mission from remote server

  • updateCount()

    Update number in left and right arrows

Properties

  • DAY_TO_MS

    Day to milliseconds
