Smartgeo (service in module smartgeomobile )

Description

Provides global methods

Methods

  • getServiceUrl(serviceName, GETParameters)

    Get ready to call URL with a service's name and list of parameters

    Parameters

    • serviceName – {string} –

      Name of called Service

    • GETParameters – {Object} –

      Associative array of get parameters name=>value

    Returns

    {string} –

    url Well formatted URL

    Example

    Smartgeo.getServiceUrl('gi.maintenance.mobility.installation.json', {
    'site'      : site.id,
    'timestamp' : site.timestamp
});

  • isRunningOnBigScreen()

    Return true if device width is >= to Smartgeo._BIG_SCREEN_THRESHOLD

    Returns

    {boolean} –

    is smartgeo running on big screen

  • isRunningOnLittleScreen()

    Return true if device width is < to Smartgeo._BIG_SCREEN_THRESHOLD

    Returns

    {boolean} –

    is smartgeo running on little screen

  • ping(callback)

    Call global.dcnx.json gimap service to know if it is reachable. So this method logout current user. It may be refactored when a real ping service will be available on gimap

    Parameters

    • callback – {function} –

      Called with boolean which depends on gimap reachability

    Example

    Smartgeo.ping(function(gimapIsReachable){
    if(gimapIsReachable){
        // do things
    }
});

  • reset()

    Clear localStorage

  • setGimapUrl(url)

    Set Gimap serveur URL on localstorage, add http:// and /index.php?service= if needed and clear localStorage

    Parameters

    • url – {string} –

      gimap server url

    Returns

    {string} –

    url setted server url

Properties

  • _BIG_SCREEN_THRESHOLD

    Width threshold which describes big screen

  • _MAX_RESULTS_PER_SEARCH

    Define max results per search (and advanced search)

  • _SMARTGEO_MOBILE_VERSION

    Smartgeo mobile version, displayed on home page
